# 🌱 SEEDS — Ideas & Projects Lab

> Wrzucasz pomysł → LUMEN go kończy.

## Format SEED

Każdy plik to jeden pomysł. Naming: `SEED_XXX_NAZWA.md`

### Statusy
- `idea` — surowy pomysł, jeszcze nie ruszony
- `in_progress` — LUMEN/SHAD pracuje
- `done` — gotowe, zdepployowane
- `paused` — odłożone

## Jak używać

1. Nowy pomysł → utwórz `SEED_XXX_NAZWA.md` z opisem
2. LUMEN dostaje SEED → generuje projekt → builder deployuje
3. Status zmienia się automatycznie
